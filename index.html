werden — present tense (full)
Foundational: builds Future tense (werden + Inf), Passive (werden + PP), and Konjunktiv II (würde).
PersonwerdenUse — FutureUse — Passive

wirwerdenWir werden lernen.Wir werden gefragt.
ihrwerdetIhr werdet lernen.Ihr werdet gefragt.
sie / SiewerdenSie werden lernen.Sie werden gefragt.
werden — Präteritum → wurde
Used for Passiv im Präteritum: wurde + Partizip II.
Personwerden (Prät.)Passive past example

Konjunktiv II — würde + Infinitiv
Default for most verbs: würde-form + Infinitiv at end. Hypothetical / polite / unreal.
PersonwürdeExample

Passiv — all tenses
werden conjugated + Partizip II at end. Agent (by whom) = von + Dativ. Means/cause = durch + Akk.
TenseStructureExample
Präsenswird / werden + PPDas Buch wird gelesen.
Präteritumwurde / wurden + PPDas Buch wurde gelesen.
Perfektist / sind + PP + wordenDas Buch ist gelesen worden.
Plusquamperfektwar / waren + PP + wordenDas Buch war gelesen worden.
Futur Iwird + PP + werdenDas Buch wird gelesen werden.
Modal + Passivmodal + PP + werdenDas muss gemacht werden.
Passiv — Aktiv ↔ Passiv transformation
AktivPassiv
Der Lehrer liest das Buch.Das Buch wird (vom Lehrer) gelesen.
Die Studentin schrieb den Brief.Der Brief wurde (von der Studentin) geschrieben.
Man hat das Haus gebaut.Das Haus ist gebaut worden.
Futur I — werden + Infinitiv
Future tense or strong assumption. Infinitiv at the end.
PersonwerdenExample
ichwerdeIch werde morgen kommen.
duwirstDu wirst es schaffen.
er / sie / eswirdSie wird studieren.
wirwerdenWir werden helfen.
ihrwerdetIhr werdet lachen.
sie / SiewerdenSie werden gewinnen.
Infinitivkonstruktionen — zu + Infinitiv
Subject must match the main clause. Separable: "auf" + "zu" + "machen" → aufzumachen.
ConstructionMeaningExample
um … zuin order toIch lerne Deutsch, um in Wien zu leben.
ohne … zuwithout doingEr ging, ohne etwas zu sagen.
statt / anstatt … zuinstead of doingStatt zu lernen, schlief er.
Es ist + adj + zu …It is … to doEs ist wichtig, viel zu üben.
Lust / Zeit haben + zuto feel like / have time toHast du Lust, ins Kino zu gehen?
vergessen / versuchen + zuto forget / try toIch versuche, früher aufzustehen.
Comparison: Konjunktiv II vs. Passiv vs. Futur
All three use forms of werden / sein / haben — easy to confuse on the exam.
FormStructureExampleMeaning
Futurwird + InfEr wird kommen.He will come.
Passiv Präs.wird + PPEs wird gemacht.It is being done.
Passiv Prät.wurde + PPEs wurde gemacht.It was done.
Konj. IIwürde + InfEr würde kommen.He would come.
Konj. II seinwäre + …Er wäre da.He would be there.
Konj. II habenhätte + …Ich hätte Zeit.I would have time.
